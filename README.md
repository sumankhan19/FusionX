**FusionX** is an AI-powered image analysis pipeline for automated and robust quantification of cell-to-cell fusion.

---

## What is FusionX?

Cell-to-cell fusion is the process by which cells merge their plasma membranes to form multinucleated syncytia. This process is fundamental to development, physiology, and disease — from placental formation and muscle development to viral infection and cancer. Quantifying fusion events typically requires specialized genetic reporters, manual counting, or labor-intensive image analysis.

FusionX changes this. It automates the entire quantification workflow using nothing more than standard **membrane** and **nuclear** fluorescent dyes — the same stains already present in most cell biology experiments. No genetic reporters, no manual counting, no subjectivity.
